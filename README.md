Heart Attack Prediction using Machine Learning

Overview

This project builds a machine learning model to predict the likelihood of a heart attack based on medical attributes such as age, cholesterol levels, blood pressure, and other clinical parameters. The goal is to demonstrate how machine learning can assist in early health risk assessment.

Dataset

The model is trained on a structured heart disease dataset containing patient health metrics and a target variable indicating the presence or absence of heart disease.

Models Used

Several machine learning algorithms were tested:
	•	Logistic Regression
	•	Random Forest
	•	Support Vector Machine (SVM)
	•	XGBoost

Results

The best performing model achieved an accuracy of ~67%, with XGBoost providing the highest performance among the tested models.
